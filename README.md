## Routed anecdotes

### Description

App where users can read and create anecdotes.

### Get started

Clone the repo:

`git clone https://github.com/lacolegiala/full-stack-routed-anecdotes`

Go to the directory and remove git:

`cd routed-anecdotes`
`rm -rf .git`

Install dependencies:

`npm install`

Run the app:

`npm start`
